Prologue

Here you can find various TileDB tutorials. You can navigate from the menu on the left of the page. Below you can find a small description of each tutorial.

TileDB Mechanics 101

It describes the main concepts of TileDB (such as arrays, tiles, etc.), as well as the physical organization of the data in persistent storage. We strongly recommend the user to carefully go through this tutorial prior to using TileDB. This will further facilitate understanding the TileDB executable and APIs, but it will also provide valuable insight into how to write super-efficient programs when using TileDB as the underlying array storage manager.

TileDB Installation

It explains how to download and install TileDB on your system, as well as how to create the C API library and use it with your code.

TileDB Executable

It provides a thorough walkthrough on how to use the TileDB executable. We strongly recommend the user to go through this tutorial immediately after finishing TileDB Mechanics 101, as it is essentially a hands-on exercise that exposes the TileDB basics.

TileDB C API

It overviews all the TileDB C API functions, including detailed examples. We strongly recommend the user to go through the TileDB Mechanics 101 and TileDB Executable tutorials, before initiating this tutorial.

TileDB Parallel Programming

It demonstrates how to write parallel programs while using TileDB as the underlying array storage manager. It includes examples of parallel filters and popular Linear Algebra computations. This is intended for users that have already mastered the TileDB Mechanics 101 and TileDB C API tutorials.

TileDB Implementation

It provides a deeper dive into the TileDB implementation, such as the most important C++ classes, the data structures and algorithms, etc. This is very useful for those interested in becoming TileDB contributors.
